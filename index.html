Subbit

Subbit creates pages of user-generated and user-ranked posts about events going on near subway stops, essentially creating a social network of off-the-radar events in NYC.

Subbit is built off a Ruby on Rails backend supported by PostgreSQL database. Currently working on getting React.js frontend running to allow native mobile app support in the future.

Visit subbit.net!
